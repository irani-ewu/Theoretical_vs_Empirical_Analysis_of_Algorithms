# Introduction
The primary goal of our CSE_246 course is to give an insight of design & analysis of the algorithms. 
Throughout the semester, we have learnt some traditional algorithms. For instance, searching 
algorithms, sorting algorithms, algorithms to find the shortest path and so on. While practicing, 
implementing these, we frequently used a term called “Time complexity”. This name refers to 
the behaviour of the certain algorithm with respect to the input size. However, that asymptotic 
analysis of any algorithm was fully theoretical, so we were heading towards something that 
would connect the theory with the practical scenario and gave us the opportunity to visualize the 
behaviour of any algorithm using our own system. This project bridges the gap between our 
conceptual understanding and applied reasoning.
